## 這是什麼

食品行銷文案的生成工具：挑好產品與想主打的賣點，就產出可直接使用的文案。

四個步驟：

| 步驟 | 說明 |
|---|---|
| 1. 選擇產品 | 從產品卡片中挑一項（湯包、冷凍水餃、即食雞胸、能量棒⋯⋯） |
| 2. 選擇賣點 | 決定文案要主打的訴求 |
| 3. 確認細節 | 微調文案的條件 |
| 4. 文案預覽 | 檢視結果並匯出使用 |
